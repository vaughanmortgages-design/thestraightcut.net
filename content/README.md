# The Straight Cut Content Studio

## Purpose

This is the master content library for:

- The Straight Cut
- eStack.ca
- eStack.us
- Vaughan Mortgage Group

## Workflow

1. Create one master article.
2. Convert it into:
   - Pinterest Pins
   - Instagram Posts
   - Facebook Posts
   - LinkedIn Posts
   - TikTok Scripts
   - YouTube Descriptions
   - Blog Articles
3. Save all content into organized folders.
4. Reuse content across brands where appropriate.
